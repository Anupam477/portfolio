# Anupam Kumar | Outstanding Creative Portfolio

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2FAnupam477%2Fportfolio)

Welcome to the source code for my professional portfolio website! It features a seamless dark-theme aesthetic, completely custom Framer Motion cursor physics, ultra-smooth scrolling, and a premium "Tech-Noir" design language.

### 🌐 Live Website: [https://portfolio-three-umber-g6h4agwn5h.vercel.app/](https://portfolio-three-umber-g6h4agwn5h.vercel.app/)

*(Note: The link above is live on Vercel)*

---

## 🚀 Technologies

- **Framework**: [Next.js 16](https://nextjs.org/) (App Directory)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: React Icons
- **Deployment**: Vercel

## 📂 Features
* **Custom Magnetic UI**: A state-of-the-art interactive custom cursor with liquid trailing and dynamic hover states over elements.
* **Premium Typography**: Metallic gradients, large striking headers, and variable fonts (Inter & Outfit).
* **Responsive Architecture**: Fully optimized for mobile, tablet, and desktop viewing.
* **Animations**: Parallax scrolling, fade-ups, and staggered element loading using optimized Framer Motion variants.
* **Lag-Free Glow Physics**: 60fps polished background blurs and ambient orbs.

## 🛠️ Running Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/Anupam477/portfolio.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the Next.js development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000)
